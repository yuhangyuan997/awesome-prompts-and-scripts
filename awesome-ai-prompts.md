# 📝 一、写作助手（22条）

## 1.1 写一篇完整的公众号文章

**使用场景：** 需要快速产出一篇结构完整的公众号推文

**提示词：**
```
你是一位资深公众号写手，擅长写10万+爆款文章。请以「{主题}」为主题，写一篇公众号文章。
要求：
- 标题要吸引眼球，包含数字或悬念
- 开头用故事或数据引入，3秒抓住读者
- 正文分成4-5个小标题段落，每段300字左右
- 语言接地气，多用排比、反问、对比
- 结尾要有金句总结+互动引导（点赞/在看/评论）
- 全文约2000字
```

**预期效果：** 获得一篇可直接发布的公众号文章，结构完整、有爆款基因

# 💻 二、编程开发（28条）

## 2.1 从零生成一个完整函数

**使用场景：** 需要实现一个特定功能但不想从头写

**提示词：**
```
请用{编程语言}实现以下功能：
功能描述：{详细描述功能}
输入：{输入格式和数据}
输出：{期望的输出格式}
约束条件：{性能要求/边界情况/限制}

要求：
- 写出完整可运行的函数/类
- 包含类型注解（如语言支持）
- 添加详细注释，解释关键逻辑
- 处理边界情况和异常
- 给出使用示例
- 说明时间复杂度和空间复杂度
```

**预期效果：** 获得可直接复制使用的生产级代码

---

## 2.2 代码调试 — 解释Bug并修复

**使用场景：** 代码报错不确定原因

**提示词：**
```
以下代码运行时报错，请帮我分析原因并修复。
编程语言：{语言}
报错信息：{粘贴报错}
代码：
```{语言}
{粘贴代码}
```

请：
1. 分析错误根因（不要只说"缺少分号"这类表层原因）
2. 给出修复后的完整代码
3. 解释为什么你的修复有效
4. 说明如何避免此类问题
```

**预期效果：** 快速定位Bug根因，获得修复方案，同时学习预防方法

---

## 2.3 代码重构 — 提升质量

**使用场景：** 代码能跑但质量差，需要优化

**提示词：**
```
请重构以下代码，目标是提升{可读性/性能/可维护性/可测试性}。
编程语言：{语言}
当前代码：
```{语言}
{粘贴代码}
```

要求：
- 保持功能完全不变
- 应用SOLID原则/设计模式（如适用）
- 提取重复逻辑，减少复杂度
- 改进命名和结构
- 附上重构前后的对比说明
- 解释每个改动的理由
```

**预期效果：** 获得高质量的代码版本，提升可维护性

---

## 10.1 链式提示（Chain Prompting）

步骤4：{优化精炼}——基于步骤3的输出
步骤5：{最终检查}——确保质量

在每个步骤开始时，请说"【步骤X开始】"，结束时说"【步骤X完成，是否继续？】"
```

**预期效果：** 通过分步骤让AI处理复杂任务，每步可控、可修改

**使用场景：** 深入思考问题，打破思维定势

**提示词：**
```
请使用苏格拉底式提问法，帮助我深入思考{主题/问题}。
我的初始观点/理解：{简要说明}

规则：
- 你不会直接给我答案
- 而是通过一系列问题引导我自己找到答案
- 每个问题都是基于我的上一个回答
- 问题会越来越深入，挑战我的假设
- 如果我的回答有逻辑矛盾，温和地指出来

请先从第一个问题开始，我会回答，你根据我的回答追问下一个问题。
对话持续5-8轮。
最后总结我从这个过程中学到了什么。
```

**预期效果：** 通过引导式提问实现深度思考和自我发现

# 🏁 附录：使用技巧

## 💡 提示词通用原则

| 原则 | 说明 |
|------|------|
| 🎭 **设定角色** | 让AI扮演特定角色（如"资深编辑""Python专家"），输出质量显著提升 |
| 🎯 **明确目标** | 告诉AI你想要什么结果，而非不要什么 |
| 📋 **提供示例** | 3个左右的输入输出示例能极大提高准确性 |
| 🔒 **增加约束** | 限制字数、格式、风格，让输出更可控 |
| 🔄 **迭代优化** | 第一次不满意就继续对话优化，而非重新开始 |
| 🧩 **复杂拆解** | 复杂任务拆成多步，让AI一步步完成 |
| ❓ **反向提问** | 让AI先问你问题了解情况，再输出 |

## 📌 常用角色设定

| 角色 | 适用场景 |
|------|---------|
| 资深编辑/撰稿人 | 写作、改写、润色 |
| 资深工程师/架构师 | 编程、技术方案 |
| 数据分析师 | 数据处理、分析报告 |
| 营销专家 | 文案、营销策略 |
| 学习导师 | 教学、答疑、规划 |
| 翻译专家 | 翻译、本地化 |
| 项目经理 | 计划书、方案 |

> 💡 **提醒**：以上提示词可直接复制使用，将 `{花括号}` 中的内容替换为你的实际需求即可。
> 有些AI模型对角色扮演特别敏感，设定角色会让输出质量大幅提升。
> 如果一次不满意，可以追加"请更详细""请更简洁""请换个角度"等指令继续优化。
